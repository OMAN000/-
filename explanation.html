مخطط الشبكة

جميع الأجهزة متصلة عبر Switch باستخدام كابلات نحاسية مباشرة.

خطوات التنفيذ (7 مراحل)

المرحلة 1: تجهيز البيئة الأساسية

  1. افتح Cisco Packet Tracer وأنشئ ملفًا جديدًا.
  2. أضف جهاز تحكم مركزي (Control PC): من End Devices → Generic PC.
  3. أضف Switch: من Network Devices → Switches → 2960 Switch.
  4. أضف أعمدة الإنارة الذكية: استخدم Generic Server لكل عمود (يمكنك تغيير الأيقونة لاحقاً).
  5. أضف كاميرا مراقبة (اختياري): استخدم IP Camera.

المرحلة 2: توصيل الشبكة

  1. اربط Control PC → Switch باستخدام Copper Straight-Through Cable.
  2. اربط كل عمود إنارة → Switch بنفس الكابل.
  3. إذا أضفت كاميرا، اربطها أيضًا بـ Switch.
  4. قم بإعطاء كل جهاز IP ثابت:
    • Control PC: 192.168.1.10
    • Lamp_1: 192.168.1.101
    • Lamp_2: 192.168.1.102
    • Lamp_3: 192.168.1.103 ... وهكذا
    • Subnet Mask: 255.255.255.0 للجميع.

المرحلة 3: إعداد أعمدة الإنارة

  1. اضغط على كل عمود → Desktop → Config → On/Off (يمثل حالة التشغيل).
  2. استخدم Simulation → Add Event لإنشاء أحداث تشغيل/إطفاء حسب جدول زمني (مثلاً تشغيل عند الغسق).

المرحلة 4: إضافة مستشعر الحركة

  1. أضف Motion Sensor أو Switch من قائمة Components لكل عمود.
  2. اربطه بالعمود بحيث: عند تفعيل المستشعر (محاكاة حركة) يضيء العمود، وبعد انتهاء الحركة ينطفئ بعد فترة.
  3. يمكن تحقيق ذلك باستخدام حدث: "Motion Detected → Turn On Lamp" و "No Motion → Turn Off after 10s".

المرحلة 5: إنشاء لوحة التحكم المركزية

  1. في Control PC → Desktop → Command Prompt، استخدم ping 192.168.1.101 للتأكد من الاتصال.
  2. يمكن إنشاء جدول تشغيل مركزي عبر Simulation → Add Event يتحكم في جميع الأعمدة.
  3. للكاميرا: يمكن ربطها بالـ PC لعرض الحركة (اختياري).

المرحلة 6: تجربة النظام

  1. فعّل Simulation Mode من شريط الأدوات.
  2. جرب:
    • تشغيل/إيقاف الأعمدة يدويًا من Control PC (باستخدام الأحداث).
    • تفعيل مستشعر الحركة → يجب أن يضيء العمود.
    • راقب استجابة الكاميرا إن وجدت.

المرحلة 7: تحسين المحاكاة

  1. أضف جدول زمني للتشغيل الليلي: من Simulation → Events، أضف حدث On/Off لكل عمود حسب الوقت (مثلاً On 6pm, Off 6am).
  2. أضف Dashboard بسيط: يمكن إنشاء صفحة HTML بسيطة على Control PC تعرض حالة الأعمدة.
  3. قسّم الشارع إلى مناطق لتسهيل الإدارة.
